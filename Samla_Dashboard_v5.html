Group 3 — Google Analytics 4 (special case)
Google Analytics 4 · samla.net
GA4 Data API v1
Free Best path: Looker iframe
  1. ★Easiest free option: open Looker Studio (free by Google) → connect GA4 property → publish a public report → embed the URL as an iframe inside the dashboard. Zero code, 5 minutes.
  2. ★For raw JSON access: Cloud Console → enable Google Analytics Data API → create a Service Account → download JSON key → run a free Cloudflare Worker as a CORS proxy.
GA4 Data API blocks browser-side requests (no CORS). Looker Studio embed is the zero-code free solution and surfaces active users, sessions, bounce rate, traffic sources, and conversions.
Group 4 — Server-side only (Python pipeline)
Why server-side: Google Ads and LinkedIn Ads APIs reject browser-origin requests (CORS) and require OAuth refresh tokens that can't safely live in client-side code. The fetcher at data_pipeline/fetch_ads.py handles both server-side and writes to live_data.json, which the dashboard reads.
Google Ads
Google Ads API v17
Manager Account required 1–2 days approval
  1. 1Create a free Manager Account (MCC) at ads.google.com/home/tools/manager-accounts
  2. 2Link your Samla Ads account to the MCC: MCC → Setup → Sub-account settings → Link existing
  3. 3Inside the MCC, go to Tools → API Center → apply for a developer token (test token issued instantly, real one within ~1 business day)
  4. 4Mint a refresh token via OAuth Playground with scope https://www.googleapis.com/auth/adwords
  5. 5Paste all five values into data_pipeline/.env
Fetches: account-lifetime spend, impressions, clicks, conversions, conversion value, CTR, CPC, ROAS. Pipeline runs on schedule; dashboard reads from live_data.json.
